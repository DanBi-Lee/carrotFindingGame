# 당근찾기 게임 🥕🥕🥕

- 게임 실행 주소 : [danbi-lee.github.io/carrotfindinggame/](https://danbi-lee.github.io/carrotFindingGame/)

## 게임 진행 방법 🥕

1. 게임 시작 버튼을 누른다.
2. 정해진 시간 내에 모든 당근을 찾으면 승리🎉

## 추가한 기능

- 화면크기에 맞춰서 게임 화면 크기 조절
- 모바일 화면 크기에 맞춰서 가로모드 제공
